# Power BI Dashboard Contest - 2024 - Chandoo.org
This repository contains the Power BI dashboard created for the Awesome Chocolates Dashboard Contest. The dashboard provides insightful visualizations on the performance of Awesome Chocolates over the last 13 months.

**Link to contest:** https://chandoo.org/wp/announcing-power-bi-dashboard-contest/


## Features

    Sales Overview: Visualizes sales trends, units sold, and profit margins.
    Shipment Analysis: Highlights shipment counts, including low-box shipments.
    Product Performance: Detailed performance metrics by product and salesperson.
    Geographical Insights: Displays sales and shipment patterns across different regions.
    Interactivity: Utilizes bookmarks, and conditional formatting.
    Drill-down Functionality: Allows detailed exploration of the data.
